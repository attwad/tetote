# Politique de retour

Nous voulons que vous soyez entièrement satisfait de votre achat chez Tetote. Si vous n'êtes pas satisfait, nous sommes là pour vous aider.

Ceci est un espace réservé pour le contenu complet de notre politique de retour. Des détails spécifiques concernant les retours, les échanges et les remboursements seront fournis ici.
